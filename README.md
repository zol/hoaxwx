# Hoax Wx

Hoax Wx is a tiny static webpage that embeds public Port Weather dashboards for locations near the Port Phillip Heads.

The page currently displays:

- Point Nepean
- Point Lonsdale
- Hovell Pile

Each dashboard is embedded as a full-width iframe and stacked vertically on the page.

## Files

- `index.html` - the static webpage containing the embedded dashboards and favicon links.
- `icon.svg` - the source wind-and-surfboard icon.
- `apple-touch-icon.png`, `favicon-32x32.png`, `favicon-16x16.png`, `favicon.ico`, `safari-pinned-tab.svg` - generated site icon assets.

## Viewing Locally

Open `index.html` directly in a browser.

## Deployment

This project is designed to be served as a static site, including through GitHub Pages. The `gh-pages` branch can be used as the publishing source with the root folder.
